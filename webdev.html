Course Description

This course is a beginner's guide to HTML and CSS, the main two programming languages used for building websites in the current technological age. In this course we will teach students the proper methods of using these two powerful languages and help young programmers build a website of their own.

Who

Students of ages 10 to 18 who

  • Have never programmed before
  • Have Minimal Experience in Programming
  • Have Minimal Experience in HTML and CSS

Structure

    • One Hour Classes
    • Zoom Virtual Classroom
    • Class Google Drive Folder
    • Assignments and Projects assigned through Google Classroom
    • Project Based Learning
    • Experienced Instructors
    • Indvidually-Tailored Interactive Learning
    • Certificate After Competiion
  • Course Information
  • 13 Weeks
  • 3 Teachers
  • 5 Projects
  • 10-15 Students per Class
