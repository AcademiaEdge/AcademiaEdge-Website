HTML/CSS

Teachers: Raghav Sriram, Avinash Valuveri, Maxwell Ye, and Eamon Mukhopadyay

This course is a beginner's guide to HTML and CSS, the main two programming languages used for building websites in the current technological age. In this course we will teach students the proper methods of using these two powerful languages and help young programmers build a website of their own. Class starts January 9th, 2021 and will fill up quickly so sign up soon!

In the first few weeks of this course we will teach students the basics of HTML using visual studio code through presentation lectures. We will then go in-depth into the styling language of CSS and help students improve their website design skills. By the end of this 15 week course taught by Raghav Sriram and Avinash Valuveri with the collaboration of Maxwell Ye, students will have built their own personalized website to show off to their friends, family, and peers.

Scratch

Teachers: Eamon Mukhopadhyay and Eric Guo

This course is a beginner’s guide to Scratch, a coding website for elementary programmers. In this course, we will teach all the basics of Scratch and help each young coder develop a passion and interest in coding. Class starts January 9th, 2021 and will fill up quickly so sign up soon!

We will first teach each component of Scratch, then have two projects at the end of the course for students to show off their knowledge. The components in Scratch include Motion and Events, Looks and Sounds, Control and Sensing, Variables and Lists, and Operators. This is a 13 week course and will be taught by Eamon Mukhopadhyay and Eric Guo.

Python

Teachers: Eamon Mukhopadhyay, Raghav Sriram, and Eric Zhu

This course is a beginner’s guide to Python, a coding language for elementary programmers. In this course, we will teach all the basics of Python, Turtle, and a brief introduction to Machine Learning and NumPy. Class starts January 9th, 2021 and will fill up quickly so sign up soon!

In the first few weeks of this course we will teach the basic of Python, slowly introducing complex functions such as the “for” loop and “while” loop. We will then transition into project based learnings having students create their own Interactive Number Guesser, creating colorful patterns using Turtle, and as the final project an input based Adventure Game.

Voices of Carmel

With the current events unfolding, it becomes increasingly difficult to voice your opinion. Voices of Carmel aims to solve just that. On this blog, all citizens of Carmel can suggest changes that will provide help to this city and share opinions on a safe space. If you have anything that you want to say about the city, please post it here and we will help you share your opinion.
