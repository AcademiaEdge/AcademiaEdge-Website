Course Description

In the advanced to C# course we will be going over migrating from console applications to making full GUI applications using WPF, writing our code in C# and writing our styles in XAML. We will go over how to make property changed events so our data on our application will auto update when the values are changed. Then we will implement local databases for data storing to be used in our final project.
